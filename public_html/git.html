Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.


Click the link below to the Git website where you can get started.

Go To Git


Here are some commands you should know:

git init - initializes a git repository
git add < filename > - adds all files to track for commit ("." includes all files)
git commit -m < message describing commit > - commits a unit of work, a complete set of changes
git checkout (-b) < branch > - switches to a branch (-b makes new branch and switches to it)

Example: git checkout -b newBranch
Creates and switches to newBranch branch

git push < remote > < branch > - pushes local commits to remote repository on github
