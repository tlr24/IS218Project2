TERMINAL

pwd - current directory
ls - list contents for current folder
cd - change directory
mkdir - make new directory
mv - move and copy file
touch - new file
vi - command/insert

GIT

git init - initialize git
git add ... ("." includes all files) - add files to local repo
git commit -m ... - commits files to remote
git checkout (-b) (-b makes new branch and switches to it) - navigate between branches
git push - push files to remote repo

Docker

docker compose up - start or restart all the services defined in a docker-compose.yml.
docker compose down - stops and removes container
docker ps - lists running containers
docker kill - shuts the container off
docker rm < container-id > - deletes the container
docker build -t < container > . - Builds a container image
docker run -dp 80:80 < container > - Runs the container in the background at the specified port numbers
