Docker is a platform that builds multi-container apps to run in any cloud environment.


Click the link below to the Docker website where you can get started.

Go To Docker


Here are some commands you should know:

docker ps - lists the running containers
docker kill < container-id > - shuts the container off
docker rm < container-id > - deletes the container

Example: docker kill 179d9f6e2b12
It will shut off the container with the container id 179d9f6e2b12

docker compose up - Builds, (re)creates, starts, and attaches to containers for a service
docker compose down - Stops containers and removes containers, networks, volumes, and images created by up
docker build -t < container > . - Builds a container image
docker run -dp 80:80 < container > - Runs the container in the background at the specified port numbers
